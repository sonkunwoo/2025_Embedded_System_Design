- 스마트 시티 구성 프로젝트 (본 이미지는 구성한 기능에 따른 상황을 표현함)

내용 : 

최근 기후변화로 인한 집중호우 및 도시 침수 피해가 빈번하게 발생하고 있으며, 특히 지하 차도와 터널 지역에서의 차량 침수 사고가 사회적 문제로 대두되고 있음. 
2023년 청주 지하차도 침수 사건으로 17대가 고립되고 많은 사상자가 발생한 재해 상황에서 실시간 모니터링과 신속한 대응 시스템의 필요성이 강조됨.
또한, 도시화가 진행됨에 따라 교통량이 증가하고 있으며, 효율적인 교통 흐름 관리와 보행자 안전 확보를 위한 지능형 교통시스템의 도입이 필수적인 상황임.

본 시스템은 라즈베리파이를 U-Boot 환경에서 신호등 제어로 사용하며, 하위 동작은 아두이노 우노를 사용하여 다양한 센서를 통해 지능형 교통시스템을 구현하였음.

### 프로젝트 성과

본 프로젝트를 통해 라즈베리파이 기반의 지능형 교통시스템 테스트베드를 성공적으로 구축하였음. 

주어진 센서류만을 활용하여 침수 감지, 적응형 신호 제어, 보행자 안전, 도로 상황 표시 등 4가지 핵심 기능을 구현하였으며, 실제 도로 환경에 적용 가능한 시스템의 테스트베드(프로토타입)을 완성하였음.
